# Skill: RAGAS Evaluation

## Metric Definitions

### Faithfulness
Measures whether the generated answer is grounded in the retrieved context.
- Score 1.0: every claim in the answer is supported by the context.
- Score 0.0: answer contains claims not present in the context (hallucination).
- Threshold: ≥ 0.80 required.

### Answer Relevancy
Measures whether the answer addresses what the user actually asked.
- Score 1.0: answer directly answers the question.
- Score 0.0: answer is off-topic or addresses a different question.
- Threshold: ≥ 0.75 required.

### Context Precision
Measures whether the retrieved chunks are actually relevant to the question.
- Score 1.0: all retrieved chunks are relevant.
- Score 0.0: retrieved chunks are irrelevant.
- Improve by: increasing score_threshold, enabling reranking.

### Context Recall
Measures whether all information needed to answer correctly was retrieved.
- Score 1.0: all ground truth facts are covered by retrieved chunks.
- Score 0.0: relevant chunks were not retrieved.
- Improve by: increasing top_k, reducing score_threshold.

### Answer Correctness
Measures factual accuracy of the answer against ground truth.
- Requires labeled dataset with ground_truth answers.
- Score 1.0: answer matches ground truth exactly.
- Threshold: ≥ 0.75 required.

## Dataset Format
```json
[
  {
    "question": "What is the return window?",
    "ground_truth": "Items can be returned within 30 days with receipt.",
    "source_document": "returns-policy.pdf"
  }
]
```

## RAGAS Evaluation Call
```python
from datasets import Dataset
from ragas import evaluate
from ragas.metrics import faithfulness, answer_relevancy, context_precision

dataset = Dataset.from_list([
    {
        "question": "...",
        "answer": "...",          # generated by your SLM
        "contexts": ["...", "..."],  # retrieved chunks (list of strings)
        "ground_truth": "...",    # from test dataset
    }
])

result = evaluate(dataset, metrics=[faithfulness, answer_relevancy, context_precision])
print(result)
```

## Interpreting Results
- faithfulness < 0.80: review system prompt; add "ONLY use context" instruction.
- context_precision < 0.70: lower score_threshold or increase top_k.
- context_recall < 0.70: increase top_k or reduce score_threshold.
- answer_relevancy < 0.75: review system prompt; ensure query is passed clearly.
- answer_correctness < 0.75: check ground truth quality; review chunk coverage.
